# 袖珍简化字拼音
基於佛振「袖珍简化字拼音」方案，額外增加了以詞定字功能。
以詞定字功能基於Lua腳本實現。

在輸入詞組過程中，按下「Shift + /」（即「?」）便可以啓動這一功能。該功能會將列表中所有詞組的每一個單字列出，并排除掉重複單字。如圖所示。
